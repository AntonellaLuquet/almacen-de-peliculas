# Introducción

Este proyecto tiene como propósito principal el almacenamiento y gestión de colecciones de películas. A través de esta aplicación, los usuarios pueden agregar, editar, y eliminar películas de su colección, así como consultar información relevante sobre cada título.

# Objetivos
1. Permitir a los usuarios registrar sus películas de forma sencilla.
2. Facilitar la organización de la colección por géneros, años y directores.
3. Proporcionar una interfaz amigable para la administración de la colección.
4. Implementar funciones de búsqueda y filtrado para una experiencia de usuario óptima.
5. Asegurar que la base de datos sea persistente y robusta.
